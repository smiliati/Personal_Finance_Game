# Personal_Finance_Game
Final project for the personal finance course. Decision game.

# Game Overview
Our personal finance game is an individual decision-making game. The game consists of questions based on all of the topics taught in class, and each question has three possible choices that the player can make. The player starts with the following stats: $2,000 Cash (USD), $500 Savings (USD), $1,000 Debt (USD), 650 Credit Score (points), 0 Stress (arbitrary unit). While the user plays the game, these stats will update based on the financial choices the player makes. The effects of the player’s decisions on these stats can be positive, negative, or neutral.

The goals of the game are to (1) show how certain financial decisions affect the stats being kept track of, and (2) to end the game with high values in the Cash, Savings, and Credit Score categories, with low values in the Debt and Stress categories.
